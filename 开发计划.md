# DuckStudy 开发计划

## 项目概述
DuckStudy是一个基于前后端分离架构的学习平台，提供课程评价、二手市场、论坛等功能。

## 技术栈
- 前端：HTML5 + CSS3 + JavaScript（原生）
  - Bootstrap 5
  - Bootstrap Icons
  - ES6 模块化
- 后端：Python + Flask
  - RESTful API
  - 模拟数据存储
  - GitHub API集成
